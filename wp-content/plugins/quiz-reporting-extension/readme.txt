== Description ==

The Quiz Reporting Extension for LearnDash exports quiz results to allow LMS website owners to analyse quiz responses at a quiz, as well as quiz taker level.


== Installation ==

Important: This plugin is a premium extension for the LearnDash plugin. You must have the LearnDash plugin already installed.

1. Upon purchasing the Quiz Reporting extension, an email will be sent to the registered email id, with the download link for the plugin and a purchase receipt id. 

2. Download the plugin using the download link.

3. Go to Plugin-> Add New menu in your dashboard and click on the ‘Upload’ tab. Choose the ‘quiz-reporting-extension.zip’ file to be uploaded and click on ‘Install Now’.

4. After the plugin has installed successfully, click on the Activate Plugin link or activate the Quiz Reporting extension from your Plugins page.

5. An Quiz Reporting Extension License sub-menu will be created under Plugins menu in your dashboard. Click on this menu and enter your purchased product’s license key. Click on Activate License. If license is valid, an ‘Active’ status message will be displayed, else ‘Inactive’ will be displayed.

6. Upon entering a valid license key, and activating the license, the results export option will be added to every quiz and quiz-taker entry (at a per quiz level).

7. For User Guide and FAQ's please refer to this link:  http://wisdmlabs.com/quiz-reporting-extension-learndash/
